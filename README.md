# TypeScript

## 类型
- 布尔基元类型对应于类似命名的JavaScript基元类型，并表示true或false的逻辑值。
<br/>
例子
```例子
var b: boolean;         // Explicitly typed  
var yes = true;         // Same as yes: boolean = true  
var no = false;         // Same as no: boolean = false
```
